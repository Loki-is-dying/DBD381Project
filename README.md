# DBD381Project

## Getting started


# For Development purposes
### starting backend
must set up env variables on first pull

    cd ecommerce-backend
    npm install
    npm start

### starting frontend
    cd ecommerce-dashboard
    npm install
    npm start

# For Viewing purposes
### starting docker
be in root directory
### build container
    docker compose up
### Remove container
    docker compose down
